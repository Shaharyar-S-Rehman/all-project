    Find the value of a 2 + b 2 when a+b=9 and a-b= 7

    Solution

    We know that

    2(a 2 + b 2 ) = (a+b) 2 + (a-b) 2

    Now putting the value of a+b and a-b

    2(a 2 + b 2 ) = (9) 2 +(7) 2

    2(a 2 + b 2 ) = 81 + 49

    2(a 2 + b 2 ) = 130

    (a 2 + b 2 ) = 130 / 2

    (a 2 + b 2 ) = 65 Ans

    Qno 2.

    Find the value of 4ab and ab when a+b=6 and a-b= 3

    Solution

    We know that

    4ab = (a+b) 2 - (a-b) 2

    Now putting the value of a+b and a-b

    4ab = (6) 2 +(3) 2

    4ab = 36 + 9

    4ab = 45 1 st Ans

    Now we find ab

    ab = 45 / 4

    ab = 11.25 Ans

    Define chemistry? Write the name of branches?

    Ans:

    The science that deals with the properties, composition, and structure of substances (defined as elements and compounds), the transformations they undergo, and the energy that is released or absorbed during these processes.

    Branches Name:

    1. Organic Chemistry
    2. Inorganic Chemistry
    3. Analytical Chemistry
    4. Physical Chemistry
    5. Biochemistry
